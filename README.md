# 🖐️ Blank Space

A hand-tracked, Iron-Man-inspired holographic OS you control with your webcam — a place to vent, play, and unwind. **Point, pinch, swipe, and fist.**

**▸ Live:** https://sharonxu-wav.github.io/blank-space/
**▸ Breakdown:** https://sharonxu-wav.github.io/blank-space/breakdown.html

One webcam, two-hand tracking, six hands-free apps:

| App | Gesture |
|-----|---------|
| 🎵 Music | pinch-spin the carousel · point to play/pause |
| 🤖 Interactive 3D | move your hand |
| 🎛️ Gesture FX | CDJ-style two-handed audio FX (filter · echo · delay · reverb · flanger · roll) |
| 🎹 Chord Lab | aim two wheels · pinch to play a piano chord |
| 😂 Meme of the Day | point & hold for the next meme |
| 🎨 Air Canvas | *coming soon* — air-doodle physics sandbox |

Works best in **Chrome or Edge** with a webcam. Tip: add `?windows` to the URL for a draggable floating-window menu instead of the carousel.

This repo hosts the built site that GitHub Pages serves — it's the compiled output, not the working source.
